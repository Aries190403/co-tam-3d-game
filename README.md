# Cô Tấm: Hành trình về làng

Prototype game 3D bằng Babylon.js và Vite.

## Gameplay

- Điều khiển Cô Tấm bằng WASD hoặc phím mũi tên.
- Giữ Shift để chạy nhanh.
- Nhặt đủ 8 con cá phát sáng.
- Né quái vật tuần tra; người chơi có 3 tim.
- Chạy qua cổng đỏ trước khi đồng hồ 3 phút kết thúc.
- Nhấn R để chơi lại.

## Chạy trên Windows

Yêu cầu Node.js 22 trở lên.

```bash
npm install
npm run dev
```

Sau đó mở địa chỉ Vite hiển thị, mặc định là `http://localhost:5173`.

## Build

```bash
npm run build
```

Repo có kèm:

- `cotam_game.html`: bản HTML độc lập.
- `media/cotam_gameplay.mp4`: video gameplay.
- `screenshots/`: ảnh preview và ảnh từ video.
